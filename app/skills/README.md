# `app/skills` — agent skills (`SKILL.md`)

One folder per skill (e.g. `search-jobs/`, `tailor-cv/`, `draft-cover-letter/`) with:

- `SKILL.md` — declaration (inputs, outputs, allowed tools).
- Optional code or prompts used only by that skill.

Reference style: <https://skills.md> (as in the hackathon brief).
